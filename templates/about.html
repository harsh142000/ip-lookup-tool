About IP & URL Lookup Tool

Welcome to the IP & URL Lookup Tool — your one-stop utility for analyzing and investigating suspicious IP addresses and URLs.

  • Dual Input Support: Accepts both IP addresses and URLs.
  • IP Intelligence: Get ISP, ASN, country, and geolocation info.
  • URL Scanning: Integrated with VirusTotal API for malicious detection.
  • IP Resolution from URLs: Automatically extracts and enriches resolved IPs.
  • Strict Validation: Filters duplicates, private, invalid IPs (up to 100 entries).
  • Summaries: Provides lookup and API usage summary separately.
  • Detailed Table Output: ASN, ISP, detections, IP origin, and more.
  • Copy & Download: Clipboard and Excel export functionality.
  • Theming: Smooth toggle between dark and light modes.
  • Modern UI: Neon borders, transitions, responsive layout.

This tool is ideal for:

  • Cybersecurity researchers & SOC teams
  • Threat investigators & analysts
  • Network admins & developers
  • Anyone validating suspicious IPs or links
